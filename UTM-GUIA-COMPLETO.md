# UTM Guia Completo — Bubbles Enterprise

> **O que é UTM?** Parâmetros que rastreiam de onde vem cada visitante do seu site.
> Quando você compartilha um link com `?utm_source=facebook&utm_medium=post`, o Google Analytics sabe: "este visitante veio do Facebook, através de um post".

---

## 🚀 Começo Rápido

### Opção 1: Copiar/Colar URLs Prontas (MAIS FÁCIL)

## 📊 Convenção de Nomes para Campanhas

Use nomes **simples e descritivos** para `[CAMPANHA]`:

| Tipo | Exemplos |
|------|----------|
| Sazonal | `spring_2026`, `summer_sale`, `holiday_promo` |
| Serviço | `emergency_repair`, `new_construction`, `free_estimate` |
| Evento | `project_showcase`, `customer_review_campaign` |
| Tipo de conteúdo | `before_after`, `educational_tip`, `case_study` |
| Período | `march_2026`, `q1_2026` |

**Bom:**
```
spring_promo_2026
emergency_repair_march
case_study_commercial
```

**Evitar:**
```
campanha123
promo_blah
xyz
```

## 🔗 Montar a URL (Passo a Passo)

1. **Escolha a plataforma** (Facebook, WhatsApp, etc)
2. **Escolha o tipo** (post, story, ad, etc)
3. **Escolha o nome da campanha** (spring_promo_2026)
4. **Copie a URL pronta acima** e substitua `[CAMPANHA]`
5. **Cole no seu post/mensagem**
6. **Pronto!** GA4 vai rastrear automaticamente

---

## 📈 Ver Resultados no GA4

Depois que os links começarem a gerar tráfego:

1. Acesse: **Google Analytics 4 → Aquisição → Resumo da campanha**
2. Procure por suas campanhas em "Campanha"
3. Veja quantos cliques/sessões vieram de cada canal
4. Veja qual canal gera mais contatos (qualify_lead)

## 🤖 Para Devs: Gerar URLs Programaticamente

Se quiser automatizar, use a função `generateUTMUrl()` em `src/lib/utm-generator.ts`:

```typescript
import { generateUTMUrl, UTM_TEMPLATES } from '@/lib/utm-generator';

// Exemplo 1: Usar template pronto
const fbUrl = generateUTMUrl(
  'bubblesenterprise.com',
  UTM_TEMPLATES.facebook.post('spring_promo_2026')
);
// → https://bubblesenterprise.com?utm_source=facebook&utm_medium=post&utm_campaign=spring_promo_2026

// Exemplo 2: Parâmetros customizados
const customUrl = generateUTMUrl('bubblesenterprise.com', {
  source: 'whatsapp',
  medium: 'direct_share',
  campaign: 'emergency_repair',
  content: 'customer_list_march' // opcional
});

// Exemplo 3: Gerar todas as variações
const allVariations = generateUTMVariations('bubblesenterprise.com', 'spring_2026');
console.log(allVariations.facebook_post);
console.log(allVariations.whatsapp);
```
